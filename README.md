# 🚀 프로젝트 소개 및 주요 기능
### 이 프로젝트는 사용자들에게 효율적인 시간 관리 및 목표 달성을 지원하는 웹 서비스입니다. 특히, 공부 시간 등록 및 관리, 목표 설정, 그리고 성취도 시각화에 중점을 두고 있습니다.

# 🌟 주요 페이지 구성 및 기능
# 🏠 랜딩 페이지 (Landing Page)
## 목적: 사이트의 첫인상 제공 및 주요 기능 안내

# 핵심 기능

- 사이트 리뷰: 사이트에 대한 전반적인 평가 및 소개를 제공합니다.

- 사용자 인증: 로그인 및 회원 가입 기능을 제공하여 사용자 접근을 관리합니다.

- 사용자 리뷰 보기: 이 사이트를 사용한 사람들의 실제 리뷰를 모아 볼 수 있습니다.

- 고객 문의: 고객문의 버튼 클릭 시 문의 사항을 작성할 수 있는 페이지로 이동합니다. 작성이 완료되면 별도의 고객 문의 관리 페이지로 이동하여 관리됩니다.

# 💻 메인 페이지 (Main Page)
## 목적: 사용자의 시간 관리 및 목표 달성 현황을 한눈에 보여주고, 주요 기능을 실행하는 중심 공간

# 핵심 기능

- 오늘 할 일 리스트: 사용자가 추가한 '오늘 할 일' 목록을 표시하며, 시작 시간(몇 월 몇 일 몇 시) 정보를 함께 제공하여 시간 계획을 쉽게 파악할 수 있습니다.

- 등록하기/시간 기록: '등록하기' 버튼을 통해 공부/작업 시간을 기록할 수 있으며, 기록된 시간에 따라 목표 달성률이 자동으로 계산되어 채워지는 기능이 포함되어 있습니다.

- 목표 설정 (Goal Setting): 사용자가 달성하고자 하는 목표를 설정하고 관리할 수 있습니다.

- 달력 (Calendar): 시간 관리 및 기록 내용을 달력 형태로 시각화하여 보여줍니다.

- 상점 (Store): 유료 기능을 확인하고 구매할 수 있는 상점 페이지로 이동합니다.

- 로그아웃 (Logout): 로그아웃 버튼 클릭 시 랜딩 페이지로 돌아갑니다.

# 🛒 상점 페이지 (Store Page)
## 목적: 유료 기능 및 서비스를 소개하고 구매를 유도

# 핵심 기능

- 유료 기능 모음: 무료 체험에서는 제공되지 않는 다양한 추가 기능들을 구매할 수 있도록 모아놓은 공간입니다.

# 📝 레지스터 페이지 (Register Page - 시간/목표 등록)
## 목적: 사용자가 집중할 시간을 구체적으로 설정하고 등록

# 핵심 기능

- 공부 시간 설정: 사용자가 얼마나 공부할 것인지 목표 시간을 정하고 등록할 수 있는 기능입니다. 이 정보가 메인 페이지의 목표 달성률 계산에 활용됩니다.

# 👥 회원 가입 페이지 (Sign-Up Page)
## 목적: 신규 사용자 계정 생성

# 핵심 기능

- 회원 가입 처리: 사용자 정보를 입력받아 계정을 생성하는 기본 회원가입 기능입니다.

# 🔑 로그인 페이지 (Login Page)
## 목적: 기존 사용자 인증 및 서비스 접근 허용

# 핵심 기능

- 로그인 처리: 사용자 아이디와 비밀번호를 통해 사용자를 인증하고 서비스에 접근하도록 하는 기본 로그인 기능입니다.

# 🛠️ 기술 기반 (Technology Foundation)
- 이 사이트가 무엇을 기반으로 운영하는 사이트인지 알려주고 이 서비스는 시간 관리 및 목표 달성을 핵심 가치로 하며, 사용자들의 학습 및 작업 효율을 극대화하는 데 중점을 둔 사이트입니다. 사용자 경험을 고려하여 직관적인 인터페이스와 실시간 목표 달성률 피드백을 제공하는 것을 기반으로 운영됩니다.
